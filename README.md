# Console-File-Manager

Консольный файловый менеджер, позволяющий:
```
1. Просмотреть список функций
2. Просмотреть список файлов и папок директории
3. Создать файл
4. Создать папку
5. Удалить файл или папку
6. Скопировать файл или папку
7. Сменить директорию
8. Сыграть в игру "Загадай компьютеру число"
```

# Архитектура системы
```
core.py                       -- модуль, содержащий в себе выполняемые функции
main.py                       -- модуль, позволяющий через терминал работать с файловым менеджером
log.txt                       -- логирование действий
game_guess_number.py          -- модуль с игрой "Загадай компьютеру число"
__init__.py                   -- маркер пакета
__pycache__                   -- системный файл
```
# Начало работы
```
1. Запустить в терминале файл main.py
2. Просмотреть список возможных команд: python main.py help
2. Выполнить необходимые команды: python main.py <command> <attributes(if it necessary)>
